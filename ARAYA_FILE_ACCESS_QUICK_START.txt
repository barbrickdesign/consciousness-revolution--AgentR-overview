BACKUPS:

--------------------------------------------------------------------------------

AUDIT LOG:

Location: 100X_DEPLOYMENT/.araya_edits.log
Format:   JSON lines (one per operation)

View in terminal:
    tail -50 100X_DEPLOYMENT/.araya_edits.log

View programmatically:
    logs = get_logs(limit=100)

Each log entry contains:
    - timestamp: When operation happened
    - type: SYSTEM, READ, WRITE, ROLLBACK, BACKUP
    - operation: SUCCESS, BLOCKED, ERROR
    - target: File affected
    - metadata: Details (size, backup path, error, etc.)

--------------------------------------------------------------------------------

ERROR HANDLING:

All functions return (success, ...) tuples:

    success, content, msg = read("file.html")
    if success:
        print(content)
    else:
        print(f"Error: {msg}")

    success, msg = write("file.html", content)
    if not success:
        print(f"Write failed: {msg}")

Common errors:
    - "Access denied: Path matches forbidden pattern: *.py"
    - "Access denied: File type not in whitelist"
    - "Access denied: Path outside allowed directory"
    - "Backup creation failed - write aborted for safety"
    - "No backups found"
