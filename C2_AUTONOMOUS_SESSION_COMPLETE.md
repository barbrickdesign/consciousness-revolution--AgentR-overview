## How to Use These Systems

### Start Background Operations
```bash
# Start scheduler daemon (runs continuously)
python BRAIN_SCHEDULER.py start

# Or run all tasks once
python BRAIN_SCHEDULER.py once
```

### Run Tasks Through Brain
```bash
# Advanced mode (9 agents)
python AUTONOMOUS_TASK_RUNNER.py run "Your task here"

# Quick mode (4 agents)
python AUTONOMOUS_TASK_RUNNER.py quick "Simple task"

# Research mode (no execution)
python AUTONOMOUS_TASK_RUNNER.py research "Research topic"
```

### Check System Health
```bash
# Full health check and auto-heal
python SELF_HEALING_SYSTEM.py auto

# Just check status
python SELF_HEALING_SYSTEM.py status
```

### Resolve Foundational Issues
```bash
# See all issues
python RECURSIVE_TASK_ENGINE.py status

# Resolve one issue
python RECURSIVE_TASK_ENGINE.py resolve missing_backup_system

# Resolve all issues
python RECURSIVE_TASK_ENGINE.py resolve-all
```
