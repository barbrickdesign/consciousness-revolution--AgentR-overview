Password Manager Setup

1

A password manager is your digital vault. It stores ALL your passwords securely so you only need to remember ONE master password.

Install Bitwarden (Free & Open Source)

  1. Go to bitwarden.com/download
  2. Download for your platform (Windows, Mac, or phone)
  3. Create a free account with your email
  4. Set a STRONG master password (16+ characters, mix of everything)
  5. Install the browser extension for Chrome/Firefox/Edge
Your master password should be a passphrase like: "correct-horse-battery-staple-42!" - easy to remember, hard to crack.
  • Downloaded Bitwarden
  • Created account
  • Set master password
  • Installed browser extension

Two-Factor Authentication (2FA)

2

2FA adds a second layer of security. Even if someone gets your password, they can't log in without your phone.

Set Up Authenticator App

  1. Download an authenticator app on your phone:
    • Google Authenticator (Android)
    • Google Authenticator (iPhone)
    • Or use Microsoft Authenticator, Authy, etc.
  2. Go to your most important accounts (Gmail, bank, social media)
  3. Find "Security" or "2FA" settings
  4. Scan the QR code with your authenticator app
  5. Save backup codes in Bitwarden!
ALWAYS save your backup codes. If you lose your phone without them, you could be locked out forever.
  • Downloaded authenticator app
  • Enabled 2FA on Gmail/Google
  • Enabled 2FA on bank accounts
  • Saved backup codes in Bitwarden

Browser Organization

3

Your browser is your window to the internet. Let's organize it for maximum efficiency.

Organize Bookmarks (7 Domains Style)

  1. Open your bookmark manager (Ctrl+Shift+O in Chrome)
  2. Create 7 folders:
    • 1_COMMAND - Control panels, dashboards, admin
    • 2_BUILD - Development, tools, projects
    • 3_CONNECT - Email, social, communication
    • 4_PROTECT - Security, banking, legal
    • 5_GROW - Business, revenue, analytics
    • 6_LEARN - Courses, docs, research
    • 7_TRANSCEND - Meditation, consciousness, wellness
  3. Sort all existing bookmarks into these folders
  4. Delete bookmarks you haven't used in 6 months
The 7 domains structure works everywhere - files, bookmarks, even physical spaces. Once you learn it, organization becomes automatic.
  • Created 7 bookmark folders
  • Sorted existing bookmarks
  • Deleted old unused bookmarks
  • Pinned most-used tabs

File Organization

4

Apply the same 7 domains structure to your files. Never lose a document again.

Create 7 Domain Folders

  1. Go to your Documents folder (or Desktop)
  2. Create these 7 folders:
    • 1_COMMAND - Daily tasks, checklists, status
    • 2_BUILD - Projects, code, creative work
    • 3_CONNECT - Contacts, letters, communications
    • 4_PROTECT - Legal docs, insurance, security
    • 5_GROW - Business plans, finances, investments
    • 6_LEARN - Notes, courses, research
    • 7_TRANSCEND - Journal, meditation, personal growth
  3. Move existing files into appropriate folders
  4. Add a README.txt in each folder explaining what goes there
When you download a file, immediately move it to the right domain folder. This habit will save you hours of searching later.

Passkey Setup

5

Passkeys are the future of authentication - no passwords at all! They use your device's biometrics (fingerprint/face) to log you in securely.

Set Up Passkeys on Key Accounts

  1. Start with Google: Google Passkeys
  2. Click "Create a passkey"
  3. Use your device's biometrics to confirm
  4. Repeat for: Microsoft, Apple, GitHub, Discord, etc.
  5. Bitwarden can also store passkeys in the vault!
Passkeys are tied to your device. Make sure you have backup access methods (2FA codes, backup email) before relying solely on passkeys.
  • Created passkey for Google
  • Created passkey for Microsoft
  • Tested passkey login
  • Kept backup 2FA active

Export & Backup Passwords

6

Don't let your passwords be trapped in one place. Regular backups ensure you never lose access.

Export from Browser to Bitwarden

  1. In Chrome: Settings > Passwords > Export passwords
  2. In Bitwarden: Tools > Import data
  3. Select "Chrome (csv)" as format
  4. Upload the exported file
  5. Verify all passwords imported correctly
  6. Delete the CSV file (it's unencrypted!)

Create Encrypted Backup

  1. In Bitwarden: Tools > Export vault
  2. Choose ".json (Encrypted)" format
  3. Save to a USB drive or secure cloud folder
  4. Store USB in a safe place (not near your computer)
Set a calendar reminder to backup your vault every 3 months. Future you will thank present you.
  • Exported passwords from browser
  • Imported into Bitwarden
  • Created encrypted backup
  • Deleted unencrypted exports

Taskbar & System Tray Cleanup

7

A clean taskbar means a clear mind. Remove the clutter and keep only what you use daily.

Clean Up Taskbar (Windows)

  1. Right-click taskbar > Taskbar settings
  2. Turn OFF: News & interests, Search box (use icon), Task view button
  3. Unpin apps you don't use daily
  4. Pin your 7 most-used apps (one per domain is ideal)

Clean System Tray

  1. Click the ^ arrow in system tray to see hidden icons
  2. Right-click programs you don't need running
  3. Choose "Exit" or "Quit" for unnecessary apps
  4. Go to Settings > Apps > Startup and disable unused apps
Less running = faster computer. If you haven't clicked an icon in a week, you probably don't need it running at startup.

  • Removed unused system tray apps
  • Disabled startup programs
  • Rebooted and verified faster startup
