‚ú® How It Works

The system maintains an in-memory cache of all inventory items and refreshes it every 5 seconds. When scanning:

  1. Each detected item is checked against the inventory database
  2. Items matching existing inventory (by type/category) are marked as "SCANNED"
  3. Purple border with glow effect is applied to scanned items
  4. New items continue to use the green/orange/red color scheme based on confidence
  5. A visual legend below the scanner explains all the colors
