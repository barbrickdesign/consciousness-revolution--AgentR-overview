Integration Guide

Basic Usage

Add the container div and include the widget script. It auto-initializes on DOMContentLoaded.

Configuration

Pass options to ArayaWidget.init() to customize position, API endpoint, and behavior.

API Methods

expand(), collapse(), send(msg), clearChat(), getState()

Persistence

Chat history is stored in localStorage by default. Disable with persistChat: false

// Basic integration
<div id="araya-widget"></div>
<script src="js/araya-widget.js"></script>

// Custom configuration
<script>
  ArayaWidget.init({
    position: 'bottom-right',
    apiEndpoint: '/.netlify/functions/araya-chat',
    enableScanlines: true,
    persistChat: true
  });
</script>
