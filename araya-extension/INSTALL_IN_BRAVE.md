# ARAYA Extension - Install in Brave
## Quick Setup Guide

---

## STEP 1: Open Brave Extensions

1. Open Brave Browser
2. Go to: `brave://extensions/`
3. Enable **Developer mode** (toggle in top-right)

---

## STEP 2: Load the Extension

1. Click **"Load unpacked"**
2. Navigate to: `C:\Users\dwrek\100X_DEPLOYMENT\araya-extension`
3. Select the folder
4. Click **"Select Folder"**

---

## STEP 3: Use ARAYA

**Sidepanel (Main Interface):**
- Click the ARAYA icon in toolbar
- Or use keyboard shortcut (set in extensions settings)
- The 7 Domains lobby opens in a sidepanel

**Quick Access:**
- Click the extension icon for popup
- Right-click on any webpage for context menu

---

## WHAT YOU GET

| Feature | Description |
|---------|-------------|
| **7 Domains Lobby** | Navigate your life areas |
| **Sidepanel Chat** | Talk to ARAYA without leaving current page |
| **Context Awareness** | ARAYA can see what page you're on |
| **Storage** | Preferences saved locally |

---

## MISSING ICONS?

If you see a blank icon, the extension still works!

To add proper icons later:
1. Create PNG images (16x16, 48x48, 128x128)
2. Save to `araya-extension/icons/` folder
3. Name them: araya-16.png, araya-48.png, araya-128.png
4. Reload the extension

---

## TROUBLESHOOTING

**Extension not loading:**
- Make sure Developer mode is ON
- Check for errors in extension page
- Reload the extension

**Sidepanel not opening:**
- Click the ARAYA icon in toolbar
- Make sure sidepanel permission is enabled
