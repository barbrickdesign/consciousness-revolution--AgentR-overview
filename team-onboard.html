Join Brave Sync Chain
2 minutes

This gives you all team bookmarks instantly.

  1. Open Brave Settings → Sync
  2. Click "I have a Sync Code"
  3. Enter the 25-word code from your admin
  4. Wait for bookmarks to appear
Open Sync Settings
Need the Sync Code?

Contact your admin for the 25-word sync code. This is shared privately, not stored here.

Bitwarden stores all team passwords securely.

  1. Create a free account
  2. Use a STRONG master password
  3. Write it down somewhere safe!
  4. Verify your email
Create Bitwarden Account

Install Bitwarden Extension
2 minutes

The browser extension auto-fills passwords.

  1. Add Bitwarden to Brave
  2. Click puzzle piece icon in toolbar
  3. Pin Bitwarden to toolbar
  4. Log in with your account
Add to Brave

  1. Check your email for Bitwarden invite
  2. Click "Join Organization"
  3. Refresh Bitwarden extension
  4. You should see shared Collections!
