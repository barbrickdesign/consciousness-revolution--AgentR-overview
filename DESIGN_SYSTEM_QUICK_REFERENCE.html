Getting Started (3 Steps)

Step 1: Add CSS to <head>
<link rel="stylesheet" href="sacred-theme.css">
Step 2: Add Home Button (after <body>)
<a href="index.html" class="home-button">← Sacred Lobby</a>
Step 3: Wrap Content
<div class="container" style="padding: 6rem 0;"> <h1 class="text-gold-gradient text-center">Title</h1> <!-- Your content --> </div>

Spacing Scale (4px grid)

Token Value Pixels Use For

var(--space-2) 0.5rem 8px Between related items
var(--space-4) 1rem 16px Component padding
var(--space-6) 1.5rem 24px Between components
var(--space-8) 2rem 32px Between sections
